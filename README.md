# Java Stopwatch

A simple stopwatch built with Java to practice programming fundamentals.

## Features
- Minute and second counter
- Real-time update in terminal
- Time formatting (00:00)

## Technologies
- Java

## What I practiced
- for loops
- nested loops
- Thread.sleep()
- System.out.printf()
- string formatting

## How to run

1. Clone the repository:

```bash
git clone https://github.com/stefanibergerdev/java-stopwatch.git
```

2. Open the project in IntelliJ IDEA or another Java IDE.

3. Run the main class.
